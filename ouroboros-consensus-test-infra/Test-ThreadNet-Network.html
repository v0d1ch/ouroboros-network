runThreadNetwork ∷ ∀ m blk. (IOLike m, MonadTimer m, RunNode blk, TxGen blk, TracingConstraints blk, HasCallStack) ⇒ ThreadNetworkArgs m blk → m (TestOutput blk) Source #

Setup a network of core nodes, where each joins according to the node join plan and is interconnected according to the node topology

We run for the specified number of blocks, then return the final state of each node.

  • tnaEpochSize ∷ EpochSize

    Epoch size

    The ThreadNet tests need to know epoch boundaries in order to know when to insert EBBs, when delegation certificates become active, etc. (This is therefore not related to the chunking of the immutable DB.)

    This is temporary: once we have proper support for the hard fork combinator, EpochInfo must be derived from the current ledger state.
